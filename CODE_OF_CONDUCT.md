# Code of Conduct

Caelo CMS adopts the **[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct, verbatim and unmodified.

The full text is canonical at the link above; we don't redistribute a copy here so the document stays in sync with the upstream project.

## Scope

The Covenant applies in all project spaces and when an individual is officially representing the project in public — that includes this repository, our discussions, issues, pull requests, the docs site, our security disclosure channels, and any community space we operate.

## Reporting

To report a Code of Conduct violation, contact **conduct@caelo-cms.com**. Reports are treated confidentially and reviewed by the project maintainers. Investigation outcomes follow the Covenant's enforcement guidelines.

If the violation involves a maintainer, you may also escalate via **[GitHub Private Vulnerability Reporting](https://github.com/caelo-cms/caelo-cms/security/advisories/new)** so the report bypasses the maintainer in question.

## Why this license

The Contributor Covenant is licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). It's the standard choice for OSS projects, has clear enforcement guidelines, and is recognised by contributors who've worked in OSS before — they know what they're agreeing to.
